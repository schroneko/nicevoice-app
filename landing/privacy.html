プライバシーポリシー ユーザーのデータをどう扱うか

NiceVoice は、音声入力を速く安全に使えることを重視しています。このページでは、収集する情報、ローカル保存の内容、外部サービスへの送信、保護方法を説明します。

1. 基本方針

NiceVoice は、必要最小限の情報のみを取り扱います。音声認識は、選択したエンジンに応じて、Mac 上のローカル処理または外部 API 経由で実行されます。利用者が選択していない外部音声認識サービスへ音声データを送信することはありません。

Apple SpeechAnalyzer、Voxtral Local、Qwen3 ASR を利用する場合、音声処理は主にローカル環境で行われます。Deepgram を選択した場合に限り、音声データは Deepgram に送信されます。

2. 収集する情報

アカウント情報

X ログイン後のユーザー名、購読状態、セッション情報、端末識別子を認証のために利用します。

アプリ利用データ

ショートカット設定、辞書設定、フィラー除去設定、言語設定、履歴、利用統計などをローカルに保存します。

音声データ

通常のリアルタイム入力では、録音データはセッション処理のためにメモリ上で扱われます。履歴に音声を残す機能を使った場合は、端末上に保存されることがあります。

ログ情報

デバッグログ、モデル起動ログ、エラー内容などをローカルのログファイルへ保存する場合があります。

3. 利用目的

  • 音声入力、文字起こし、辞書変換、句読点補正の提供
  • サブスクリプション状態と端末登録状態の確認
  • ローカルモデルのダウンロードとローカルサーバーの起動
  • 不具合調査、セキュリティ対策、品質改善
  • 利用者からの問い合わせへの対応

4. 外部送信・第三者提供

nukosuku.com ログイン状態、購読状態、端末登録の確認に利用します。
Deepgram Deepgram エンジンを利用した場合に限り、音声データと認識リクエストを送信します。
Hugging Face Voxtral / Qwen3 ASR のローカルモデルを取得する際に利用します。
YouTube / yt-dlp YouTube 文字起こし機能を利用した場合に、対象 URL に対して音声取得処理を実行します。

法令に基づく場合を除き、利用者の個人情報を、上記の処理提供に必要な範囲を超えて第三者へ提供することはありません。

5. 保存と削除

  • 認証トークンや端末識別子は Keychain を使用して保存します。
  • 辞書設定、履歴、利用統計、UI 設定は主に UserDefaults またはアプリのローカル保存領域に記録されます。
  • 保存済みの履歴や辞書は、アプリ内操作またはアプリ削除により端末から削除できます。
  • アカウント連携を解除した場合、保存済みの認証情報は削除されます。

6. セキュリティ

NiceVoice は、Keychain 利用、証明書ピンニング、一部ローカル署名検証などの手段で、認証情報や通信の保護を行います。完全な安全性を保証するものではありませんが、合理的な範囲で継続的に改善します。

7. お問い合わせ

本ポリシーに関する質問、保存情報の確認依頼、削除依頼は support@nicevoice.app までご連絡ください。
