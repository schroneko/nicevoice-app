2. 価格・支払方法

販売価格 購入画面または料金ページに表示する金額
追加料金 インターネット接続費用、外部 API 利用料、決済手数料等は利用者負担となる場合があります
支払方法 クレジットカードその他、購入画面に表示する決済手段
支払時期 初回購入時、またはサブスクリプションの更新日に自動決済

3. 提供時期

決済完了後、対象アカウントで利用可能となった時点またはアプリのダウンロード完了時点で提供が開始されます。外部サービス障害や審査待ち等により、反映まで時間がかかる場合があります。

4. 解約・返金

  • デジタル商品の性質上、法令上必要な場合を除き、購入後の返金には対応しない場合があります。
  • サブスクリプションの解約は、次回更新日前までに所定の方法で行ってください。
  • 外部 API キーを利用する機能の料金や返金は、各プロバイダーの規約に従います。

5. 動作環境

NiceVoice は Apple Silicon 搭載 Mac と macOS 26 以降を前提としています。ローカル ASR や YouTube 文字起こし機能には、`uvx`、`hf`、`yt-dlp` 等の追加依存が必要になる場合があります。
