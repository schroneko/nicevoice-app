2. 現在の提供条件

アプリ単体販売 現在は行っていません
利用対象 ぬこスク加入者向けの先行アクセス特典として提供しています
追加費用 インターネット接続費用や、外部 API を利用する場合の利用料は、利用者負担となることがあります
将来の有償提供 将来、単体販売や別の提供形態へ変更する場合は、本ページを更新して告知します

3. 提供時期

ぬこスク加入者として先行アクセス権が有効になり、対象アカウントでログイン確認が完了した時点で利用を開始できます。外部サービス障害や認証反映の遅延により、反映まで時間がかかる場合があります。

4. 解約・返金

  • 現在、NiceVoice 自体の単体販売やアプリ単位の返金受付は行っていません。
  • ぬこスクの加入・解約・返金条件は、当該プラットフォームまたは運用側の案内に従います。
  • 外部 API キーを利用する機能の料金や返金は、各プロバイダーの規約に従います。

5. 動作環境

NiceVoice は Apple Silicon 搭載 Mac と macOS 26 以降を前提としています。ローカル ASR や YouTube 文字起こし機能には、`uvx`、`hf`、`yt-dlp` 等の追加依存が必要になる場合があります。
