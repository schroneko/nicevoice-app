1. サービス内容

NiceVoice は、macOS 向けの音声入力・文字起こし・辞書変換・履歴管理・バッチ文字起こし等の機能を提供します。利用可能な機能や対応エンジンは、アプリのバージョン、アクセスモード、OS バージョン、外部 API の状況により変わる場合があります。

2. アカウントと認証

  • 一部機能の利用には、指定された認証フローによるログインが必要です。
  • 利用者は、自身のアカウント情報を適切に管理する責任を負います。
  • 先行アクセス権や端末登録状態の確認結果により、利用できる機能が制限される場合があります。

3. 先行配布と外部サービス

NiceVoice は現在、ぬこスク加入者向けの先行配布として提供しています。将来的に一般公開または提供形態の変更を行う場合があります。また、利用者自身が API キーを設定して使う機能については、外部サービスの利用料金、利用条件、障害対応は当該サービス提供元の規約にも従います。

Deepgram などの外部音声認識サービスを有効にした場合、その送信・利用は利用者の選択に基づいて実行されます。

4. 禁止事項

  • 法令または公序良俗に反する利用
  • 不正アクセス、逆コンパイル、脆弱性悪用、ライセンス回避の試み
  • 第三者の権利を侵害するコンテンツの作成または送信
  • サービス基盤や外部 API に過度な負荷を与える利用
  • 不正な再配布、転売、アカウント共有、認証回避

5. 提供停止・変更

NiceVoice は、保守、障害対応、法令対応、外部サービスの制約その他合理的な理由により、機能の一部または全部を変更・停止することがあります。重大な変更がある場合は、可能な範囲で更新履歴やサイト上で告知します。

6. 免責

  • 音声認識結果の完全性・正確性・無誤謬性を保証するものではありません。
  • 外部サービスの障害、ネットワーク障害、OS 仕様変更に起因する不具合について、提供者が制御できない範囲があります。
  • 法令上許される範囲で、間接損害、逸失利益、データ消失等について責任を負わないものとします。

7. 規約変更

本規約は、必要に応じて改定することがあります。改定後の規約は、ウェブサイトまたはアプリ内で掲示した時点から効力を生じます。
